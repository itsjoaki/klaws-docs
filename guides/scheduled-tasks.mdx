# Create Scheduled Tasks

Scheduled tasks let your agent work on autopilot. Here's how to set one up.

## Creating a task

<Steps>
  <Step title="Go to Tasks">
    Open your agent's dashboard and find the **Tasks** section.
  </Step>
  <Step title="Click Create Task">
    Hit the button to create a new scheduled task.
  </Step>
  <Step title="Describe what you want">
    Write a plain-English description of what your agent should do. Be specific.

    **Good:** "Search X/Twitter for the top 5 trending AI topics and write a short summary of each."

    **Too vague:** "Check Twitter."
  </Step>
  <Step title="Set the schedule">
    Pick how often the task should run:

    - **Every hour** — for things that need frequent updates
    - **Every 6 hours** — a few times a day
    - **Every day** — once per day at a set time
    - **Every week** — once per week
    - **Custom** — set your own interval
  </Step>
  <Step title="Save and activate">
    Save the task. It will start running on the next scheduled time.
  </Step>
</Steps>

## Checking results

Every time a task runs, the results show up in your [Activity feed](/features/activity). Check there to see what your agent produced.

## Managing tasks

- **Pause** — Temporarily stop a task without deleting it
- **Edit** — Change the description or schedule at any time
- **Delete** — Remove a task you no longer need

<Tip>
  Start with one or two tasks and check the Activity feed to make sure the results are what you expect before adding more.
</Tip>

<Warning>
  Each task run uses your agent's resources. Running many high-frequency tasks can add up quickly.
</Warning>
