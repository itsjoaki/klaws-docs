---
title: Activity
description: See what your agent has been up to.
---

The Activity tab shows a chronological feed of everything your agent does in the background:

- **Scheduled task runs** — see when tasks executed and what they produced
- **Telegram conversations** — view message exchanges from Telegram
- **Important events** — system events and notifications

Each entry shows the timestamp, source (scheduled/Telegram), and a preview of the output.

<Tip>
  Use Activity to check if your scheduled tasks are running correctly. If a task failed or produced unexpected results, you'll see it here.
</Tip>
