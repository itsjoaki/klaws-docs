# Quickstart

## 1. Create an account

Go to [useklaws.com/login](https://useklaws.com/login) and sign in with Google or email.

## 2. Deploy your agent

Click **Deploy Agent** from the dashboard. Your agent will be ready in seconds.

## 3. Start chatting

Type anything in the chat. Your agent will respond. Try:

- "What can you do?"
- "Search the web for the latest AI news"
- "Write me a Twitter thread about [topic]"

## 4. Customize

- **Soul** — Edit your agent's personality and instructions
- **Skills** — Browse and install from 170+ available skills
- **Channels** — Connect Telegram to chat from your phone
- **Tasks** — Ask your agent to set up scheduled tasks

## That's it

Your agent is now running 24/7. It learns from every conversation and gets better over time.
