# Welcome to Klaws

Klaws gives you a personal AI agent that learns, runs 24/7, and reaches you anywhere — web, Telegram, and more.

## What can your agent do?

- **Chat** — Talk to your agent anytime via web or Telegram
- **Learn** — Your agent remembers your preferences, gets smarter over time, and auto-learns new skills
- **Research** — Search the web, browse websites, analyze content
- **Create** — Write posts, threads, articles, newsletters
- **Automate** — Schedule tasks that run while you sleep (check X, post content, monitor things)
- **Connect** — Link your agent to Telegram, X/Twitter, and more

## How it works

1. **Sign up** — Create an account with Google or email
2. **Deploy** — Your agent spins up in seconds with its own AI brain
3. **Chat** — Start talking. The more you use it, the smarter it gets
4. **Customize** — Edit its personality (Soul), install skills, connect channels

Your agent runs on its own server with dedicated memory, skills, and personality. It's yours — nobody else can access it.
