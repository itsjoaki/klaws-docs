# Scheduled Tasks

Scheduled tasks let your agent do things automatically — even when you're not around. Set it once, and it runs on repeat.

## How it works

You give your agent:

1. **A task** — what you want it to do (in plain English)
2. **A schedule** — how often it should run (every hour, every day, every Monday, etc.)

Your agent then runs the task at the scheduled time, every time. Results show up in your [Activity feed](/features/activity).

## What can you schedule?

Pretty much anything your agent can do:

- Check X/Twitter for trending topics and summarize them
- Post content to social media at a set time
- Search the web for news about a topic and send you a summary
- Monitor a website for changes
- Generate a daily briefing every morning

## Schedule options

| Schedule | What it means |
|---|---|
| Every hour | Runs once per hour |
| Every 6 hours | Runs 4 times a day |
| Every day | Runs once per day at a set time |
| Every week | Runs once per week on a specific day |
| Custom | Set your own schedule |

<Tip>
  Start with longer intervals (daily or weekly) to see how your agent handles the task before increasing the frequency.
</Tip>

<Warning>
  Scheduled tasks count toward your agent's usage. Running many tasks every hour can use up resources quickly.
</Warning>

For a step-by-step guide, see [How to create scheduled tasks](/guides/scheduled-tasks).
