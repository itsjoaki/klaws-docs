# Deploy Your Agent

When you click Deploy, Klaws creates a dedicated environment for your agent:

- **Isolated workspace** — Your agent has its own memory, skills, and conversations
- **Always running** — A background process keeps your agent alive 24/7
- **Auto-configured** — AI model, tools, and settings are set up automatically

## What your agent gets

| Feature | Description |
|---------|-------------|
| AI Brain | Powered by state-of-the-art language models |
| Memory | Remembers everything across conversations |
| Web Access | Can search the web and browse websites |
| Skills | Can learn new skills automatically or from the marketplace |
| Scheduling | Can run tasks on a schedule (every hour, daily, etc.) |
| Channels | Accessible via web chat and Telegram |

## Agent limits

Each agent runs on shared infrastructure. Current limits:

- **Chat**: Unlimited messages
- **Scheduled tasks**: Up to 10 active tasks
- **Skills**: Unlimited installs from the marketplace
- **Memory**: Persistent and unlimited

These limits may change as we scale.
